O limite de uma sequência é um dos tópicos mais antigos de análise matemática, a qual formaliza rigorosamente o conceito de sequência convergente.

De forma intuitiva, supondo que tem-se uma sequência de pontos (por exemplo, um conjunto infinito de pontos numerados utilizando os números naturais) em algum tipo de objeto matemático (por exemplo, os números reais ou um espaço vetorial) que admite o conceito de vizinhança (no sentido de "todos os pontos dentro de uma certa distância de um dado ponto fixo"). Um ponto L é o limite da sequência se para toda a vizinhança que se defina, todos os pontos da sequência (com a possível exceção de um número finito de pontos) estão próximos a L. Isto pode ser interpretado como se houvesse um conjunto de esferas de tamanhos decrescentes até zero, todas centradas em L, e para qualquer destas esferas, só existiria um número finito de números fora dela.

Definição formal

[editar | editar código-fonte]

Para uma sequência de pontos { x n | n ∈ N } {\displaystyle \{x_{n}|n\in \mathbb {N} \}\;} em um espaço métrico M {\displaystyle M} com função de distância d {\displaystyle d} [1] (como por exemplo, uma sequência de números racionais, números reais, números complexos, pontos em um espaço normado, etc) diz-se que L ∈ M {\displaystyle L\in M} é o limite da sequência e escreve-se

L = lim n → ∞ x n {\displaystyle L=\lim _{n\to \infty }x_{n}}

se, e somente se

∀ ϵ > 0 ∃ n 0 ∈ N : n > n 0 ⇒ d ( x n , L ) < ϵ . {\displaystyle \forall \epsilon >0\;\exists n_{0}\in \mathbb {N} :n>n_{0}\Rightarrow d(x_{n},L)<\epsilon .\;}

ou seja, se, e somente se, para todo número real ϵ > 0 {\displaystyle \epsilon >0\;} , existe um número natural n 0 {\displaystyle n_{0}} tal que para cada n > n 0 {\displaystyle n>n_{0}\;} tem-se d ( x n , L ) < ϵ {\displaystyle d(x_{n},L)<\epsilon } , o que equivale a dizer que, para o caso real, dado ϵ > 0 {\displaystyle \epsilon >0} , existe N {\displaystyle N} tal que n > N ⇒ | a n − L | < ϵ ⇒ L − ϵ < a n < L + ϵ {\displaystyle n>N\Rightarrow |a_{n}-L|<\epsilon \Rightarrow L-\epsilon <a_{n}<L+\epsilon } [2].

Uma generalização desta relação, para uma sequência de pontos { x n | n ∈ N } {\displaystyle \{x_{n}|n\in \mathbb {N} \}\;} em um espaço topológico T {\displaystyle T} : diz-se que L ∈ T {\displaystyle L\in T} é um limite desta sequência[1] e escreve-se

L = lim n → ∞ x n {\displaystyle L=\lim _{n\to \infty }x_{n}}

se, e somente se, para toda a vizinhança V {\displaystyle V} de L {\displaystyle L} existe um número natural N {\displaystyle N} tal que para cada n ∈ N {\displaystyle n\in N} tem-se x n ∈ V {\displaystyle x_{n}\in V} .

Se uma sequência tem limite, diz-se que a sequência é convergente, e que a sequência converge ao limite. Caso contrário, a sequência é divergente.

Comentários

[editar | editar código-fonte]

A definição significa que eventualmente todos os elementos da sequência aproximam-se tanto como queiramos ao valor limite. A condição que impõe que os elementos encontrem-se arbitrariamente próximos aos elementos subsequentes não implica, em geral, que a sequência tenha um limite (veja sucessão de Cauchy).

É possível, também, que uma sequência em um espaço topológico geral, possa ter vários limites diferentes, mas uma sequência convergente possui um único limite se T é um espaço de Hausdorff, por exemplo, a reta real (estendida), o plano complexo, seus subconjuntos (R, Q, Z...) e produtos cartesianos (Rn...).

Exemplos

[editar | editar código-fonte]
  • A sequência (1/1, 1/2, 1/3, 1/4, …) de números reais converge ao limite 0;
  • A sequência (1, -1, 1, -1, 1, …) é divergente;
  • A sequência (1/2, 1/2 + 1/4, 1/2 + 1/4 + 1/8, 1/2 + 1/4 + 1/8 + 1/16, …) converge ao limite 1 (este é um exemplo de uma série infinita);
  • Se a é um número real com valor absoluto |a| < 1, então a sequência an possui limite 0;
  • lim n → ∞ 1 n p = 0  se  p > 0 {\displaystyle \lim _{n\to \infty }{\frac {1}{n^{p}}}=0{\hbox{ se }}p>0} ;
  • lim n → ∞ n 1 n = 1 {\displaystyle \lim _{n\to \infty }n^{\frac {1}{n}}=1} ;
  • lim n → ∞ a 1 n = 1  se  a > 0 {\displaystyle \lim _{n\to \infty }a^{\frac {1}{n}}=1{\hbox{ se }}a>0} .
  • No caso de uma sequência de números reais a n = 3 n 2 + 4 n n 2 + n − 4 , {\displaystyle a_{n}={\frac {3n^{2}+4n}{n^{2}+n-4}},} pode-se verificar, a partir da definição, que o seu limite é 3 {\displaystyle 3} . De fato, nota-se que, dado ϵ > 0 {\displaystyle \epsilon >0} ,
| 3 n 2 + 4 n n 2 + n − 4 − 3 | = | 3 n 2 + 4 n − 3 n 2 − 3 n + 12 n 2 + n − 4 | = | n + 12 n 2 + n − 4 | ≤ | n + 12 n 2 − 4 | {\displaystyle \left|{{\frac {3n^{2}+4n}{n^{2}+n-4}}-3}\right|=\left|{\frac {3n^{2}+4n-3n^{2}-3n+12}{n^{2}+n-4}}\right|=\left|{\frac {n+12}{n^{2}+n-4}}\right|\leq \left|{\frac {n+12}{n^{2}-4}}\right|}

e que, além disso,

1 2 n 2 > 4 ⇒ n > 8 ⇒ n > 3 {\displaystyle {\frac {1}{2}}n^{2}>4\Rightarrow n>{\sqrt {8}}\Rightarrow n>3} ,

de modo que

| n + 12 n 2 − 4 | ≤ | n + 12 n 2 − 1 2 n 2 | = | 2 n + 24 n 2 | ≤ | 2 n + 24 n n 2 | = | 26 n n 2 | = | 26 n | < ϵ ⇒ n > 26 ϵ {\displaystyle \left|{\frac {n+12}{n^{2}-4}}\right|\leq \left|{\frac {n+12}{n^{2}-{\frac {1}{2}}n^{2}}}\right|=\left|{\frac {2n+24}{n^{2}}}\right|\leq \left|{\frac {2n+24n}{n^{2}}}\right|=\left|{\frac {26n}{n^{2}}}\right|=\left|{\frac {26}{n}}\right|<\epsilon \Rightarrow n>{\frac {26}{\epsilon }}}

desde que n > max { 3 ; 26 ϵ } {\displaystyle n>\max\{3;{\frac {26}{\epsilon }}\}} . Em outras palavras, dado ϵ > 0 {\displaystyle \epsilon >0} , existe N = m a x { 3 ; 26 ϵ } {\displaystyle N=max\{3;{\frac {26}{\epsilon }}\}} , tal que:

n > max { 3 ; 26 ϵ } ⇒ | 3 n 2 + 4 n n 2 + n − 4 − 3 | < ϵ {\displaystyle n>\max\{3;{\frac {26}{\epsilon }}\}\Rightarrow \left|{{\frac {3n^{2}+4n}{n^{2}+n-4}}-3}\right|<\epsilon } .

Resultados para o caso de sequências de números reais

[editar | editar código-fonte]

Destacam-se os seguintes resultados para limites de sequências[3]:

  • Todo limite de uma sequência é um valor único, ou seja, se a n {\displaystyle a_{n}} tende a um número real A {\displaystyle A} e, também, a um número real B {\displaystyle B} , então A = B {\displaystyle A=B} ;
  • Se uma sequência a n {\displaystyle a_{n}} tende a L {\displaystyle L} , então toda subsequência de a n {\displaystyle a_{n}} convergirá para esse mesmo número L {\displaystyle L} ;
  • Toda sequência convergente é limitada;
  • Toda sequência monótona e limitada converge (disso, segue que se uma subsequência da sequência monótona a n {\displaystyle a_{n}} converge para L {\displaystyle L} , então a n {\displaystyle a_{n}} é convergente e tem limite L {\displaystyle L} );
  • Se uma sequência tem um limite positivo, é garantido que todos os termos desta sequência são positivos a partir de um certo índice (resultado análogo vale para o caso de a sequência ter limite negativo);
  • Para duas sequências a n {\displaystyle a_{n}} e b n {\displaystyle b_{n}} convergentes, com a n ≥ b n {\displaystyle a_{n}\geq b_{n}} , tem-se lim ( a n ) ≥ lim ( b n ) {\displaystyle \lim(a_{n})\geq \lim(b_{n})} , sendo possível substituir qualquer uma das sequências por uma constante qualquer.
  • Para sequências a n ≤ b n ≤ c n {\displaystyle a_{n}\leq b_{n}\leq c_{n}} em que a n {\displaystyle a_{n}} e c n {\displaystyle c_{n}} têm limite L {\displaystyle L} , segue que o limite de b n {\displaystyle b_{n}} é L {\displaystyle L} ;
  • Toda sequência que é limitada possui uma subsequência convergente (Teorema de Bolzano-Weierstrass).[2]

Limite superior e limite infeior

[editar | editar código-fonte]

Destacam-se os seguintes resultados para limites de sequências[3]:

Ver artigo principal: Limite superior e limite inferior

Se uma subsequência de uma sequência possui como limite um valor L {\displaystyle L} , dizemos que L {\displaystyle L} é um ponto de aderência, ou valor de aderência, ou ponto aderente da sequência.[2] O maior desses pontos aderentes é o limite superior ( lim n → ∞ sup j ≤ n a j = lim ¯ a n ) {\displaystyle \lim _{n\rightarrow \infty }\sup _{j\leq n}a_{j}={\overline {\lim }}\,a_{n})} ) e o menor é o limite inferior ( lim n → ∞ inf j ≤ n a j = lim _ a n {\displaystyle \lim _{n\rightarrow \infty }\inf _{j\leq n}a_{j}={\underline {\lim }}\,a_{n}} ).

Toda sequência limitada ( a n ) {\displaystyle (a_{n})} possui um ponto aderente máximo A {\displaystyle A} (o maior limite de uma subsequência) e um ponto aderente mínimo a {\displaystyle a} (o menor limite de uma subsequência), onde:

  • Qualquer que seja ϵ > 0 {\displaystyle \epsilon >0} , existem infinitos índices n {\displaystyle n} tais que A − ϵ < a n {\displaystyle A-\epsilon <a_{n}} e somente um número finito com A + ϵ < a n {\displaystyle A+\epsilon <a_{n}} ;
  • Qualquer que seja ϵ > 0 {\displaystyle \epsilon >0} , existem infinitos índices n {\displaystyle n} tais que a n < a + ϵ {\displaystyle a_{n}<a+\epsilon } e somente um número finito com a n < a − ϵ {\displaystyle a_{n}<a-\epsilon } .

Da mesma forma, é garantido que uma sequência limitada ( a n ) {\displaystyle (a_{n})} converge para L {\displaystyle L} se, e somente se, os limites superior e inferior são iguais a L {\displaystyle L} .

Exemplo

[editar | editar código-fonte]

Considerando a sequência a n = ( − 1 ) n = ( − 1 , 1 , − 1 , 1 , − 1 , 1 , − 1 , . . . ) , {\displaystyle a_{n}=(-1)^{n}=(-1,1,-1,1,-1,1,-1,...),} nota-se que as únicas subsequências convergentes são da forma ( − 1 , − 1 , − 1 , − 1 , − 1 , . . . ) {\displaystyle (-1,-1,-1,-1,-1,...)} ou ( 1 , 1 , 1 , 1 , 1 , . . . ) {\displaystyle (1,1,1,1,1,...)} , de modo que 1 {\displaystyle 1} e − 1 {\displaystyle -1} são os únicos pontos de aderência de a n {\displaystyle a_{n}} . Assim, lim ¯ a n = 1 {\displaystyle {\overline {\lim }}\,a_{n}=1} e lim _ a n = − 1 {\displaystyle {\underline {\lim }}\,a_{n}=-1} .

Propriedades aritméticas do limite

[editar | editar código-fonte]

Dadas duas sequências convergentes ( x n ) {\displaystyle (x_{n})} e ( y n ) {\displaystyle (y_{n})} , onde lim ( x n ) = X {\displaystyle \lim(x_{n})=X} e lim ( y n ) = Y {\displaystyle \lim(y_{n})=Y} , as sequências ( x n + y n ) , ( x n y n ) , ( k x n ) {\displaystyle (x_{n}+y_{n}),(x_{n}y_{n}),(k\,x_{n})} são convergentes, sendo k {\displaystyle k} um número real qualquer[2]. Ainda,

  1. lim ( x n + y n ) = lim x n + lim y n = X + Y {\displaystyle \lim(x_{n}+y_{n})=\lim x_{n}+\lim y_{n}=X+Y} ;
  2. lim ( k x n ) = k lim x n = k X {\displaystyle \lim(k\,x_{n})=k\,\lim x_{n}=k\,X} . Caso k = − 1 {\displaystyle k=-1} , temos que: x n → X ⇒ − x n → − X {\displaystyle x_{n}\rightarrow X\Rightarrow -x_{n}\rightarrow -X} ;
  3. lim ( x n y n ) = lim ( x n ) lim ( y n ) = X Y {\displaystyle \lim(x_{n}y_{n})=\lim(x_{n})\,\lim(y_{n})=XY}
  4. lim ( x n y n ) = lim ( x n ) lim ( y n ) = X Y {\displaystyle \lim \left({\frac {x_{n}}{y_{n}}}\right)={\frac {\lim(x_{n})}{\lim(y_{n})}}={\frac {X}{Y}}} , desde que y n ≠ 0 {\displaystyle y_{n}\neq 0} .

Limites infinitos

[editar | editar código-fonte]

Quando uma sequência segue uma regularidade de comportamento de tal forma que o termo se torna arbitrariamente grande enquanto aumenta o índice, temos que a sequência diverge para infinito positivo. Quando se torna muito pequeno enquanto aumenta o índice, a sequência diverge para infinito negativo[2] . Formalmente, uma sequência x n {\displaystyle x_{n}} diverge para + ∞ {\displaystyle +\infty } se, dado k > 0 {\displaystyle k>0} , existe N {\displaystyle N} tal que n > N ⇒ x n > k {\displaystyle n>N\Rightarrow x_{n}>k} e diverge para − ∞ {\displaystyle -\infty } se, dado k < 0 {\displaystyle k<0} , existe N {\displaystyle N} tal que n > N ⇒ x n < k {\displaystyle n>N\Rightarrow x_{n}<k} .

Os limites infinitos também possuem propriedades aritméticas:

  1. a n → + ∞ ⇔ − a n → − ∞ {\displaystyle a_{n}\rightarrow +\infty \Leftrightarrow -a_{n}\rightarrow -\infty } ;
  2. Uma sequência a n {\displaystyle a_{n}} não limitada e monótona tende a + ∞ {\displaystyle +\infty } se for não decrescente e tende a − ∞ {\displaystyle -\infty } se for não crescente;
  3. lim ( a n ) = ± ∞ ⇒ 1 a n → 0 {\displaystyle \lim(a_{n})=\pm \infty \Rightarrow {\frac {1}{a_{n}}}\rightarrow 0} ;
  4. lim ( a n ) = 0 ⇒ 1 a n → ± ∞ {\displaystyle \lim(a_{n})=0\Rightarrow {\frac {1}{a_{n}}}\rightarrow \pm \infty } se a n > 0 {\displaystyle a_{n}>0} ou a n < 0 {\displaystyle a_{n}<0} respectivamente;
  5. Se a n → ± ∞ {\displaystyle a_{n}\rightarrow \pm \infty } e b n {\displaystyle b_{n}} é limitada, então ( a n + b n ) → ± ∞ {\displaystyle (a_{n}+b_{n})\rightarrow \pm \infty } , respectivamente;
  6. Se a n → ± ∞ {\displaystyle a_{n}\rightarrow \pm \infty } e b n ≥ c {\displaystyle b_{n}\geq c} , com c {\displaystyle c} um número positivo, então a n b n → ± ∞ {\displaystyle a_{n}b_{n}\rightarrow \pm \infty } ;
  7. Se a n → ± ∞ {\displaystyle a_{n}\rightarrow \pm \infty } e b n ≤ c < 0 {\displaystyle b_{n}\leq c<0} , então a n b n → ∓ ∞ {\displaystyle a_{n}b_{n}\rightarrow \mp \infty } ;
  8. Se a n → + ∞ {\displaystyle a_{n}\rightarrow +\infty } e a n ≤ b n {\displaystyle a_{n}\leq b_{n}} , então b n → + ∞ {\displaystyle b_{n}\rightarrow +\infty } .

Critério de convergência de Cauchy

[editar | editar código-fonte]

O critério de convergência de Cauchy auxilia na verificação da convergência de uma sequência, sem fornecer o valor do limite, caso ele exista. Ele afirma que uma sequência de números reais é convergente se, e somente se, qualquer que seja ϵ > 0 {\displaystyle \epsilon >0} , exista N {\displaystyle N} tal que:

n , m > N ⇒ | a n − a m | < ϵ {\displaystyle n,m>N\Rightarrow |a_{n}-a_{m}|<\epsilon } ,

o que equivale a afirmar que, dado ϵ > 0 {\displaystyle \epsilon >0} , existe N {\displaystyle N} tal que, para todo inteiro positivo p {\displaystyle p} :

n , m > N ⇒ | a n − a n + p | < ϵ {\displaystyle n,m>N\Rightarrow |a_{n}-a_{n+p}|<\epsilon } .
